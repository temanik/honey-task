# Go Slices

## Использование

Запуск(из корня проекта):

### Запуск одной задачи
```bash
go run main.go slices 1
```
### Запуск нескольких задач
```bash
go run main.go slices 1 5 10
```
### Просмотр справки
```bash
go run main.go slices
```
